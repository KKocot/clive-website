This post kicks off the Clive blog. Expect short release notes, deeper internals
write-ups, and the occasional how-to as the wallet matures from beta toward 1.0.

## What is Clive?

Clive is a power-user wallet for the [Hive](https://hive.io) blockchain, built
around a CLI for scripts and a mouse-driven TUI for everyday operations. Keys
live inside a local, encrypted Beekeeper wallet — never plain-text on disk.

A few principles we keep coming back to:

- **Terminal-first, not terminal-only.** The TUI is fully keyboard- and
  mouse-navigable; the CLI mirrors every action for automation.
- **Encryption by default.** No `clive --insecure` mode. Beekeeper or nothing.
- **Honest about beta.** Auth setup may still change before 1.0; we'll flag
  breaking changes prominently in release notes.

## A taste of the CLI

You can list your profiles with a single command:

```bash
clive show profiles
```

Or open the TUI and start operating:

```bash
clive
```

> Beta software — verify checksums and review the install script before running
